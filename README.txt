How to compile:
    - run 'make'. 
        This will run 'gcc -o DET DET.c -g' automatically.

How to run:
    - run './DET'
        This runs the program.

How to add your own 3x3 matrix:
    - Declare and init a 2d 3x3 int array in main() and
    call 'compute_determinant()', passing the 2d array
    as parameter.
